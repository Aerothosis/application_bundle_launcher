==== Change Log
1.0.0.2 - (20221215) Create shortcut button
- Added a "Make Shortcut" button that will create a cmdline enabled shortcut for target bundle. User can also select
	a source exe/ico to use as the icon for the shortcut.

1.0.0.1 - (20221215) CmdLine Launching Options
- Added ability to auto-launch app bundle from cmd line. Requires '--"NAME"' in cmd line

1.0.0.0 - (20210918) Initial Version
- Can add application packages and containing app/url targets
- Can edit all of those
- Can launch all of those
- Can close all applications (does not close browser)
